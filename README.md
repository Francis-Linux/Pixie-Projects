# 🏦 Banking System (C++)

A simple banking system created using C++ that allows users to create accounts, deposit money, withdraw money, and check balances through a command-line interface.

---

# 📌 Features

- Create bank accounts
- Deposit money
- Withdraw money
- Check account balance
- Simple menu system
- Beginner-friendly C++ project
- Uses functions and basic OOP concepts

---

# 🛠 Technologies Used

- C++
- Standard Template Library (STL)
- Command Line Interface (CLI)

# ▶️ How to Run

## 1️⃣ Clone the Repository

```bash
git clone https://github.com/Francis-Linux/Pixie-Projects.git
```

---

## 2️⃣ Open the Project Folder

```bash
cd Pixie-Projects
```

---

## 3️⃣ Compile the Program

Using g++:

```bash
g++ banking_system.cpp -o banking_system
```

---

## 4️⃣ Run the Program

### Windows

```bash
banking_system.exe
```

### Linux / macOS

```bash
./banking_system
```
